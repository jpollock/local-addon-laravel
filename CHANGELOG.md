## [0.0.1] - 2024-12-08

### Added

- Initial release
- One-click Laravel site creation from Local's site wizard
- Automatic environment configuration (database, APP_URL, APP_KEY)
- Artisan command integration panel
- Quick command buttons for common tasks (migrate, cache:clear, etc.)
- Laravel Breeze starter kit support
- Custom Nginx configuration for Laravel routing
- Site header and sidebar badges to identify Laravel sites
- VS Code, Site folder, and Site shell integrations opening Laravel root

### Technical

- Full TypeScript implementation with strict mode
- Zod validation for all IPC inputs
- Jest test infrastructure with mocks
- CI/CD with GitHub Actions
